# Koa 2 Restful Boilerplate

## Description
Koa 2 RESTful API using :

 - Koa 2
 - Mongodb + Mongoose
 - Babel
 - Asynchronous Functions (Async/Await)

## Running
Install dependencies
```
npm install
```

Start a Mongodb instance
```
mongod
```

Start a local server
```
npm start
```

Run test
```
npm test
```
